Developed a real-time style data analytics project by cleaning and analyzing retail sales data using Python and SQL, and building an interactive Power BI dashboard to visualize KPIs, revenue trends, customer segments, and regional insights for business decision-making.
This project focuses on analyzing retail sales data to uncover key business insights related to revenue, profit, customer behavior, and regional performance. The goal is to simulate a real-world data analyst workflow by performing data cleaning, transformation, analysis, and building an interactive dashboard for decision-making.
**Tools & Technologies**
Python (Pandas) → Data cleaning & preprocessing
SQL (MySQL) → Data analysis & querying
Power BI → Interactive dashboard & visualization

**Data Processing Steps**
-->1. Data Cleaning (Python)
Handled missing values
Removed duplicates
Converted date columns to datetime format
Created new features:
Year
Month
-->2. Data Storage (SQL)
Loaded cleaned dataset into MySQL database
Created structured table for analysis
Performed SQL queries for business insights
-->3. Data Analysis (SQL)
Key analysis performed:
Year-wise sales trend
Category-wise revenue contribution
Region-wise performance
Top customers by revenue
Average order value
Discount vs Profit analysis
